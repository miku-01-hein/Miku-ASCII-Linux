# 新版本v2.01现于2025/12/6发布
- 提升了编译性能
- 更饱和的色彩
- 优化了算法(避免不必要的计算)
- 减少单函数过多if esle

脚本 `start.sh` 仅适用与RHEL、Debian、Arch系
其他发行版请自行安装或手动编译所需依赖

# 使用方法

 - chmod +x start.sh
 - ./start.sh

  当你运行这个`Shell`脚本后, 它将会为您更新操作系统的所有包
然后自动拉取编译所需要的依赖包, 并自动编译`C++`代码最后输出并运行启动.

你也可以手动换为其他mp4来观赏它的ASCII编码mp4
需到 `start.sh` 中第197行中
将  `./miku miku.mp4 ascii.mp4 150 1.5` 中的miku.mp4修改
再到第218行中
将 `mpv miku.mp4 & mpv ascii.mp4` 中的miku.mp4修改
